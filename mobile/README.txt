Pixel Bird mobile offline edition

Files:
- index.html

How to open on Android:
1. Send this folder or pixel-bird-mobile.zip to the phone.
2. Unzip it.
3. Tap index.html.
4. Choose a browser such as Chrome, Edge, or the system browser.

How to open on iPhone:
1. Send pixel-bird-mobile.zip to the phone.
2. Save it to the Files app and unzip it.
3. Tap index.html.
4. If it opens in a preview that blocks sound or touch, share/open it with Safari.

Controls:
- Tap the game screen to fly.
- Tap the top button to start or restart.
- The game is portrait-friendly and does not require internet.

Notes:
- Some chat apps do not run local HTML directly in their preview. Save the file first, then open it with a browser.
- Sound starts only after the first tap because mobile browsers require user interaction before audio.
