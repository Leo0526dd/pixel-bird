# 中国大陆可访问部署方案

目标：让用户不用下载、不解压、不使用 VPN，直接打开网址玩 Pixel Bird。

## 推荐方案：腾讯云 CloudBase 静态网站托管

适合这个游戏，因为 Pixel Bird 是纯静态网页。

步骤：

1. 登录腾讯云控制台。
2. 开通 CloudBase / 云开发。
3. 进入静态网站托管。
4. 上传 `pixel-bird-static-site.zip` 解压后的全部文件，或上传 `out` 目录里的全部文件。
5. 默认首页设置为 `index.html`。
6. 发布后使用 CloudBase 提供的网址访问。

优点：

- 中国大陆访问更稳定。
- 不需要自己维护服务器。
- 可以后续绑定自定义域名。

## 备选方案：腾讯云 COS 静态网站

步骤：

1. 创建 COS 存储桶。
2. 上传 `out` 目录全部文件。
3. 开启静态网站设置。
4. 首页文档设置为 `index.html`。
5. 如果需要自己的域名，再绑定自定义域名并配置 HTTPS。

## 备选方案：阿里云 OSS 静态网站

步骤：

1. 创建 OSS Bucket。
2. 上传 `out` 目录全部文件。
3. 开启静态网站托管。
4. 首页设置为 `index.html`。
5. 如果要作为正式网站使用，建议绑定自定义域名和 HTTPS。

## 备选方案：国内服务器

如果你有腾讯云、阿里云、华为云等国内服务器：

1. 安装 Nginx。
2. 把 `out` 目录上传到网站根目录。
3. Nginx 指向该目录。
4. 配置域名和 HTTPS。

## 当前项目产物

运行：

```bash
npm run build
```

会生成：

```text
out/
```

这个目录就是完整静态网站。上传 `out` 里的所有文件即可。

也可以直接使用：

```text
pixel-bird-static-site.zip
```

解压后上传里面的全部文件。
