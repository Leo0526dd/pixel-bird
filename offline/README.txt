Pixel Bird offline edition

How to play:
1. Send pixel-bird-offline.html to another device.
2. Double-click or tap the file to open it in a browser.
3. No GitHub, VPN, server, or internet connection is required.

Controls:
- Click or tap to fly.
- Space / Enter / Arrow Up to fly.
- Esc to pause or resume.

Notes:
- Sound starts after the first user click/tap because browsers require user interaction before audio playback.
- High score is saved in the local browser on each device.
